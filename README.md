# Personal Portfolio Website

https://maximechasles.github.io

## Bonjour, soyez les bienvenus !

Je m'appelle Maxime, j'ai repris mes études en Informatique après avoir travaillé près de 10 ans dans la restauration.

## Objectif

Ce projet est destiné à mettre à en pratique les compétences Web acquises durant mes années de formation. 

Il s'agit de créer un espace doté d'une page d'accueil menant vers un résumé de mon parcours, un aperçu de mes autres projets et une page de contact. 

## Stack

+ HTML5
+ CSS3
+ Javascript

## Hébergement

J'ai choisi d'utiliser [GitHub Pages](https://docs.github.com/fr/pages/getting-started-with-github-pages/about-github-pages) qui me permet d'héberger mon site gratuitement à partir du dépôt.

+ [Création d’un site GitHub Pages](https://docs.github.com/fr/pages/getting-started-with-github-pages/creating-a-github-pages-site)
+ [Configuration d’un domaine personnalisé sur GitHub Pages](https://docs.github.com/fr/pages/configuring-a-custom-domain-for-your-github-pages-site)

## Organisation

+ **Homepage** - Message de bienvenue et menu de navigation.
+ **About** - Bref résumé de mon parcours, photo, téléchargement de mon CV.
+ **Projects** - Galerie présnentant mes autres projets.
+ **Contact** - Formulaire de contact et liens LinkedIn, GitHub, GitLab.
